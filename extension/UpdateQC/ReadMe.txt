Modify the following parameters in the run.bat in order to update the case status on QC

@userName: It is the user name that uses to log in to the QC
@project: It is the QC project
@testFolder: It is the QC test plan path
@testSetName: It is the test set name in the test folder
@testFilePath: It is the result file path

Please also modify the test result file according to the template file (TestSuites.xls)
